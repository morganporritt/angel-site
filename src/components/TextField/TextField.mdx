# TextField

A labeled text input for forms. It renders a `<label>` wired to the `<input>`
via `id`/`htmlFor`, with optional helper (`hint`) text. All native input
attributes pass through (`type`, `placeholder`, `value`, `onChange`,
`disabled`, `required`, `autoComplete`, …).

```tsx
import TextField from "@/components/TextField";

<TextField
  id="signup-email"
  label="Email address"
  type="email"
  placeholder="you@example.com"
  hint="We'll never share your email."
/>;
```

## Guidelines

- Always provide an `id` so the label is programmatically associated with the
  input — this is required for accessibility.
- Prefer a visible `label` over placeholder-only fields; placeholders disappear
  on input and aren't a substitute for labels.
