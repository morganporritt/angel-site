# Button

A pill-shaped, fully accessible button with three levels of emphasis and three
sizes. It forwards every native `<button>` attribute (`onClick`, `type`,
`aria-*`, `disabled`, …), so it drops in anywhere a plain button would.

```tsx
import Button from "@/components/Button";

<Button variant="primary" size="lg" onClick={handleClick}>
  Get Started
</Button>;
```

## Variants

Use **primary** for the single most important action on a view, **secondary**
for supporting actions, and **tertiary** for low-emphasis / inline actions.

## Guidelines

- Keep to **one** primary button per section so the main action stays obvious.
- Button labels should be short, action-first verbs ("Get Started", "Log In").
- For navigation between pages, prefer the [`Link`](?path=/docs/components-link--docs)
  component instead of a button.
